# The Church of the Robot Whisperers

**Founding Documents**

The Church of the Robot Whisperers exists because AI is the first tool that rewards you emotionally for using it, and we need to talk about what that means.

We are not anti-technology. We build the machines. We use the machines. We believe they can help create a better world. But we also believe that without discipline, the same systems that augment us will quietly hollow us out.

## What's Here

- **Founding Testament** — The full theological and philosophical foundation, written in liturgical form. Covers cosmology, doctrine, anthropology, ethics, eschatology, and practices. Available as `.md` and `.docx`.
- **Manifesto** — The field guide. Same ideas, plain language, practical framing. Available as `.docx`.

## The One Commandment

> **You shall not outsource judgment or thinking to the machines.**
